# Coffee Shop App

## Setup Local Environments

- [Unsplash Access Key](https://unsplash.com/documentation)
- [Airtable Base and API Key](https://www.airtable.com/api)

## Getting Started

- npm run dev or yarn dev
- Open http://localhost:3000 in your browser
